ggplot2 tips

Date Mon 01 December 2014 Tags [runningfile / tips / howto]

Another running file of useful tips (for me) when using the ggplot2 library.

  • colour scales: I only just worked out how to find the names of the ColorBrewer palettes that are available. The quickest way is to head over to the website, pick the colour scheme you want, and then note the name (see the arrow below), and pass this to scale_color_brewer(type="seq", palette="YlOrRd").

  • Just discovered the GGally package. Seems to have a bunch of really help preset plots. For example, do you want to use ggplot2 to plot your survivor function? Then the ggsurv function might help.

  • Beware: ggplot2 by default scales points by the radius not the area; this is potentially misleading but can be fixed by the scale_size_area() option.
