How to report frailty in a Cox Model?

Thu 27 March 2014
By Steve Harris

Struggling to report frailty in a survival model. Any ideas? I posted the question to Cross Validated.

Update 2015-12-11

There's an answer now for parametric models, but below in the end I used the Median Hazard Ratio1 (analogous to the Median Odds Ratio).

Here's a little code snippet to estimate the median hazard ratio in Stata (after stcox).

stcox $indep_vars , nolog shared(group_var) noshow
di "Variance of random effects is e(theta) = " + `=e(theta)'
* So to estimate the MHR you now need to
local twoinvtheta2 = 2 / (e(theta)^2)
local mhr = exp(sqrt(2*e(theta))*invF(`twoinvtheta2',`twoinvtheta2',0.75))
di "MHR: `mhr'"
