Setting up SSH keys for a Synology NAS

Date Fri 30 March 2018 Tags howto

Wow. What an absolute pain. Getting my synology NAS to work with SSH keys. In the end the majority of the instructions link2 here seemed to work.

Plus link1 for the port change

So ...

On the NAS you need to modify /etc/ssh/sshd_config and uncomment lines as below replacing the port number with your custom number.

source
RSAAuthentication yes
PubkeyAuthentication yes
AuthorizedKeysFile  .ssh/authorized_keys
port 22

On your machine you need to run ssh-keygen -t rsa and accept the defaults.

Then copy that using (replace 12345 with your port number and user_name and nas_name as you'd expect)

ssh -p12345 user_name@nas_name.local "/bin/cat >> ./.ssh/authorized_keys" < ~/.ssh/id_rsa.pub
